## Run Locally

Clone the project

```bash
  git clone git@github.com:Hamada-Ali/Bank-React-App
```

Go to the project directory

```bash
  cd Bank_Modern_App
```

Install dependencies

```bash
  npm install
```

Start the server ( The App run on Vite )

```bash
  npm run dev
```
